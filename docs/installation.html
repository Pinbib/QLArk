Installation ​

QLArk is available on Windows, Linux, MacOS:

OS ​

OSFILETESTEDDOCS
WindowsHERE✅DOCS
LinuxHERE❌DOCS
MacOSHERE❌DOCS

Window ​

To work with QLArk, download this file to your device and move this file to the C:/Windows/System32/ folder. Then follow the steps here.

Linux ​

To work with QLArk, download this file to your device. Then, for this, you need to transfer this file to the /usr/bin/ folder to do this, open a terminal and run:

bash
sudo cp QLArk /usr/bin/
bash
sudo chmod +x /usr/bin/QLArk

Then follow the steps here.

MacOS ​

To work with QLArk, download this file to your device. Then, for this, you need to transfer this file to the /usr/bin/ folder to do this, open a terminal and run:

bash
sudo cp QLArk /usr/bin/
bash
sudo chmod +x /usr/bin/QLArk

Then follow the steps here.

After ​

After installation, open a terminal and run the following command:

bash
qlark version

You should get something like:

text
QLArk: v0.0.0
